## Start with IDX Guides
Identity Experience Framework is a ready-to-use solution for all the necessary actions. It is a set of registration, authentication, and related web pages such as forgot password, profile. 

It is hosted at LoginRadius, and you can perform all the actions on the Identity Experience Framework page without making any explicit configurations. Click [here](/guides/idx-overview/) for Detailed Overview and customization of IDX.
